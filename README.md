# 时光策展案 - 印象派记忆画廊

一个温馨的互动网页，五卷温情档案，表面是悬疑解谜，翻开是友情故事。

## 项目简介

这是一个为闺蜜准备的生日惊喜网页，通过收集5个线索解锁温暖真相。网页采用莫奈印象派风格设计，背景是动态水彩效果，照片保持原样清晰展示。

## 功能特点

- 🎨 **莫奈风格背景**：Canvas 绘制的动态水彩睡莲池效果
- 🖼️ **原图展示**：五张案例照片保持原样，仅加画框装饰
- 🔄 **3D翻转卡片**：点击卡片查看案件真相
- ⭐ **线索收集**：点击线索按钮收集星星，5个线索解锁最终报告
- 🎁 **最终祝福**：集齐线索后查看合成祝福图

## 案件列表

1. **CASE-001** 狮子的守护契约 - NUS毕业照
2. **CASE-002** 第七个座位的预留 - 聚餐拍立得
3. **CASE-003** 彩虹伞的时空坐标 - 重庆街拍
4. **CASE-004** 甜心病毒的源代码 - 甜酷自拍
5. **CASE-005** 第零号案件：花束的时区 - 毕业花束

## 交互流程

```
正面：照片 + 正面故事 + [寻找线索]按钮
    ↓ 点击[寻找线索]
弹出线索弹窗（1.5秒后自动翻转）
    ↓ 自动翻转
背面：真相档案
    ↓ 点击卡片
返回正面
```

## 技术栈

- Next.js 16 + React + TypeScript
- Tailwind CSS
- Framer Motion（动画）
- Lucide React（图标）

## 安装运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 照片替换

将照片放入 `public/images/` 目录：
- `case-001.jpg` - CASE-001 照片
- `case-002.jpg` - CASE-002 照片
- `case-003.jpg` - CASE-003 照片
- `case-004.jpg` - CASE-004 照片
- `case-005.jpg` - CASE-005 照片
- `final-gift.png` - 最终祝福合成图

## 部署

构建后的 `dist` 目录可直接部署到 Vercel、Netlify 或任何静态托管服务。

```bash
npm run build
# 部署 dist 文件夹
```
